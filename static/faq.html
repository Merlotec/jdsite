How can I prevent my children changing ProdiG settings?

ProdiG control panel requires administrator privilages to change ProdiG settings. Therefore, your child should not be an administrator of the machine, which will prevent them from changing settings.

Why aren't my websites/programs being blocked?

Ensure that you have the mode set to Blacklist in the Applications tab, and either disable the Schedule or check to to make sure your schedule is correct. If the schedule is enabled, ProdiG will only block apps/websites if the current time is specified in the schedule. ProdiG is in alpha phase and there will be bugs - if you still can't fix the problem be sure to contact us and we'll help you out.

Do you have a Mobile App?

Unfortunately no - mobile operating systems do not give Apps the ability to interact with/shut down other apps, or block websites. However iOS and Android both have their own parental control/usage limitation settings built in to the operating system.

How does ProdiG block websites?

ProdiG installs an HTTP system proxy to detect and block access to websites. Most windows apps will use the system proxy when accessing websites. However, browsers can be programmed to use custom proxies to bypass the system proxy.
