How quickly can you run one mile? Choose a safe place to run a mile or do it on a treadmill and measure a mile. Make sure you and your parents agree that it is a safe 'place and time' to run. Aim to run at least twice a week. Over two months keep a record of your timed runs. Work hard to see how much you can improve your time.
You will need to create a table in which you should record each run carefully.
